Logical Star Schema Model Configuration

Fact_Employees Table
  • Keys: EmployeeID, JoinDateKey
  • Attributes: Gender, Designation, Salary, Status, AttendanceRate, SatisfactionScore, PerformanceScore
Dim_Department Table
  • Attributes: DepartmentID, DepartmentName
  • Relationship: 1-to-many relationship with Fact_Employees
Dim_Calendar Table
  • Attributes: DateKey, CalendarDate, CalendarMonth, Year
  • Relationship: 1-to-many relationship linking with Employee JoinDate

Power Query Data Transformation (M Code)

let
    Source = Folder.Files("C:\HR_Data\"),
    #"Import_Dataset" = Excel.Workbook(File.Contents("C:\HR_Data\image_0b1eb9.xlsx"), null, true),
    FactEmployees_Sheet = #"Import_Dataset"{[Item="FactEmployees",Kind="Sheet"]}[Data],
    #"Promoted Headers" = Table.PromoteHeaders(FactEmployees_Sheet, [PromoteAllScalars=true]),
    #"Changed Type" = Table.TransformColumnTypes(#"Promoted Headers",{
        {"EmployeeID", type text}, 
        {"EmployeeName", type text}, 
        {"Gender", type text}, 
        {"Department", type text}, 
        {"Designation", type text}, 
        {"JoinDate", type date}, 
        {"Salary", Int64.Type}, 
        {"Status", type text}, 
        {"AttendanceRate", type number}, 
        {"SatisfactionScore", type number}, 
        {"PerformanceScore", type number}
    }),
    #"Replaced Errors" = Table.ReplaceErrorValues(#"Changed Type", {{"AttendanceRate", 0}, {"SatisfactionScore", 0}})
in
    #"Replaced Errors"

Structured DAX Measures

Copy formulas below directly to DAX interface
DAX Measure 1. Total Headcount 
Total Headcount = 
DISTINCTCOUNT('Fact_Employees'[EmployeeID])
DAX Measure 2. Attendance Index % 
Attendance Index % = 
AVERAGE('Fact_Employees'[AttendanceRate])
DAX Measure 3. Recruitment Stream (YTD) 
New Joins YTD = 
CALCULATE(
    COUNT('Fact_Employees'[EmployeeID]), 
    YEAR('Fact_Employees'[JoinDate]) = 2025
)
DAX Measure 4. Resigned Counter 
Resigned Counter = 
CALCULATE(
    COUNT('Fact_Employees'[EmployeeID]), 
    'Fact_Employees'[Status] = "Resigned"
)
